# Paint

Projeto 1 da cadeira de Computação Gráfica - Universidade Federal do Cariri

## Atalhos do Teclado

| Tecla | Ação                 |
| ----- | -------------------- |
| `p`   | Criar Ponto          |
| `l`   | Criar Linha          |
| `g`   | Criar Polígono       |
| `d`   | Deletar Objeto       |
| `r`   | Rotacionar           |
| `s`   | Selecionar           |
| `t`   | Transladar           |
| `+`   | Aumentar Escala      |
| `-`   | Diminuir Escala      |
| `x`   | Refletir             |
| `c`   | Cisalhar (Shear)     |
| `F8`  | Salvar Dados         |
| `F9`  | Carregar Dados       |
| `Esc` | Modo de Visualização |
